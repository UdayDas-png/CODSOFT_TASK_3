# 🧮 Calculator Web App

This repository contains a simple and functional calculator web application built using HTML, CSS, and JavaScript.

---

## 📁 Project Files

* **calculator.html**
  Main HTML file that provides the structure of the calculator interface.

* **style.css**
  Stylesheet used to design and layout the calculator UI.

* **script.js**
  JavaScript file that handles the calculator logic and user interactions.

* **UdayDas-png**
  Image asset used in the project (if applicable).

* **README.md**
  Documentation file describing the project.
